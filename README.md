# 3. 기능구현 및 역할 분담

- 공통 header/footer
- 스플래시

**⚒️ 기능구현**

- 회원가입

#### 서동현
**🎨 UI**
- 메인 페이지
- 책 상세정보

**⚒️ 기능구현**
- 메인 피드게시물 리스트업
- 유저 검색 / 다른 유저 프로필 페이지 이동
- Trend Book
- 책 상세정보 모달구현

<br />

#### 양혜진
**🎨 UI**
- 프로필 수정
- 책 검색 모달

**⚒️ 기능구현**
- 프로필 수정

<br />

#### 이승미
**🎨 UI**
- 공통컴포넌트 제작(button, input, modal, 기타 공통 요소)
- 마이페이지 / 다른사람 프로필
- 내 서재
- 팔로워/팔로잉 리스트

**⚒️ 기능구현**
- 팔로우/언팔로우
- 게시물 좋아요
- 로그아웃
- 게시물 작성 통계 - Nivo chart

#### 최강우

- 게시물 등록
